What is ParSolGen?

ParSolGen (Parallel Solvers Generator) simplifies high-performance computing by allowing scientists and engineers to write high-level descriptions of numerical algorithms. It automatically generates efficient parallel C++ code, taking care of the complexities of distributed memory, load balancing, and accelerator usage.

Automated Distributed Memory

Seamless support for distributed memory systems without manual MPI programming.

Load Balancing

Automatic workload distribution across computing nodes to maximize efficiency.

Accelerator Support

Out-of-the-box Goopax support that allows employig accelerators in the computation.
